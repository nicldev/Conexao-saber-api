# Evidências da Validação - RedaIA

Esta pasta contém as evidências fotográficas e em vídeo da validação realizada com o público-alvo específico.

## Conteúdo

### Fotografias
- `foto_apresentacao_01.jpg` - Apresentação do projeto
- `foto_apresentacao_02.jpg` - Demonstração das funcionalidades
- `foto_feedback_01.jpg` - Coleta de feedback
- `foto_equipe_01.jpg` - Equipe de desenvolvimento

### Vídeos
- `video_apresentacao.mp4` - Gravação completa da apresentação
- `video_demonstracao.mp4` - Demonstração das funcionalidades principais

## Autorizações

✅ Todas as fotografias e vídeos foram obtidos com autorização explícita dos participantes.

✅ Consentimento formal foi registrado para uso acadêmico das imagens.

## Observações

- As imagens foram editadas para proteger a privacidade quando necessário
- Nomes e informações pessoais foram removidos ou ofuscados quando aplicável
- Todos os participantes assinaram termo de consentimento

## Uso

Estas evidências podem ser utilizadas para:
- Comprovação da validação realizada
- Apresentações acadêmicas
- Documentação do projeto
- Melhorias futuras baseadas nas interações observadas

---

**Nota:** Se as evidências não puderem ser incluídas no repositório por questões de privacidade, deve-se criar um documento alternativo descrevendo as evidências e onde elas estão armazenadas.
